git操作的具体步骤 1、cd 路径 进入当前目录 2、配置git基本操作 【没有消息就是好消息】 3、git init 在本地进行初始化（建立暂存区） .git文件存储当前项目的所有版本信息 4、工作区 => 暂存区 git add 文件名 git add * 提交所有文件 git commit -m "这一次提交的描述" 5、查看当前工作区的状态 git status 6、从暂存区恢复到本地区 git restore/checkout 文件名 7、查看工作区和暂存区版本的区别 git diff 8、clear 清屏操作 9、git log 查看已经提交到暂存区的版本 10、恢复文件到指定的某一个版本 git reset --hard 版本号 11、生成ssh密匙 ssh-keygen -t rsa -C "你的github邮箱" 12、github账户配置密匙 13、暂存区 => 远程仓库 git remote add origin "https//github.com/LORELEl/testDemo.git" git push -u origin master 【注】提交过程可能需要输入账户和密码 14、
